# 车辆配置（导航 / 运动规划 / 控制器 调参）

> 逐一核对自 `small_brain/config/nav/*.yaml`、`small_brain/config/sim_controllers.yaml`、
> `arm/mini_mec_six_arm_moveit_config/config/*.yaml`。本文档汇总当前使用的导航、运动规划与控制器参数。
> 本文档管「软件可调参数」；硬件描述见《03_车辆本体》，建图见《02_RTAB-Map》。

导航主力：**move_base + 全局 GlobalPlanner + 局部 TEB**。地图来自 RTAB-Map 的 `/rtabmap/grid_map`。

## 4. 代价地图

### `costmap_common_params.yaml`（全局/局部共用）

| 层 | 关键参数 |
|----|---------|
| footprint | [[−0.143,−0.115],[−0.143,0.115],[0.143,0.115],[0.143,−0.115]] |
| static_layer | `map_topic=/rtabmap/grid_map`, **track_unknown_space=true**, trinary=true |
| obstacle_layer | 源 `depth_scan`：topic=`/scan`, sensor_frame=`camera_depth_optical_frame`, obstacle_range=8.0, raytrace_range=8.5, clearing+marking |
| inflation_layer | cost_scaling_factor=5.0, **inflation_radius=0.2** |

> 障碍来自深度相机转 `/scan`（`depthimage_to_laserscan`），无实体雷达。

### `global_costmap_params.yaml`

- 静态全图 20×20 m，resolution **0.05**，update/publish 1.0 Hz。
- plugins：static_layer + inflation_layer。
- **inflation_radius=1.1**，cost_scaling_factor=8.0（路径尽量走走廊中央、离墙远）。

### `local_costmap_teb_params.yaml`（TEB 主流程）

- 滚动窗 4×4 m，resolution 0.05，update **5.0** Hz / publish 2.0 Hz。
- plugins：static + obstacle + inflation。
- inflation_radius=0.25，cost_scaling_factor=5.0；obstacle_layer sensor_frame=`camera_link`。

> TEB 主流程使用 `local_costmap_teb_params.yaml`；`dwa_local_planner_params.yaml` 保留为配置文件，但主流程未启用。

## 5. 控制器（`sim_controllers.yaml`）

| 控制器 | 类型 | 关节 | PID |
|--------|------|------|-----|
| joint_state_controller | JointStateController | — | publish_rate 50Hz |
| **arm_controller** | `position_controllers/JointTrajectoryController` | joint1–5 | p=100, i=0.01, d=1.0, i_clamp=1.0 |
| **hand_controller** | `position_controllers/JointTrajectoryController` | joint6–11 | 同上 |

> 手臂与夹爪均使用 `position_controllers/JointTrajectoryController`，URDF transmission 均为 `PositionJointInterface`（见《03_车辆本体》§4）。
> constraints：goal_time=2.0, stopped_velocity_tolerance=0.1, 各关节 goal=0.05。

## 6. MoveIt 运动规划（`arm/mini_mec_six_arm_moveit_config/config/`）

| 文件 | 关键值 |
|------|--------|
| `kinematics.yaml` | KDL 求解器；search_resolution=0.005；**timeout=0.1**；attempts=10；**position_only_ik=true**；base=base_link，tip=link5 |
| `joint_limits.yaml` | joint1–11 全部 max_velocity=10, max_acceleration=10 |
| `ompl_planning.yaml` | 23 种规划器配置（RRT/RRTConnect/PRM…），默认 RRTConnect |
| `ros_controllers.yaml` / `sensors_3d.yaml` / `chomp_planning.yaml` | MoveIt 侧控制器与可选 CHOMP/3D 感知配置 |

> IK timeout 为 **0.1 s**，并启用 `position_only_ik`（只解位置不解姿态）。
